这是一个小游戏，名字为 《猜单词》。
> 规则：
> 答案为指定长度单词，输入对应长度单词即可;
> 绿色代表此单词中有此字母且位置正确;
> 黄色代表此单词中有此字母，但该字母所处位置不对
> 灰色代表此单词中没有此字母
> 一共有单词长度+1次机会，机会用尽前猜中即为胜利
